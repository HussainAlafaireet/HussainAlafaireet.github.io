Project Overview

How can we extend the specifications of CSS to control the look, behavior, and position of this AR content in an intelligent way?

Role: UX Researcher  |  Duration: 4 Months
Tools Used: AFrame Web AR Framework, Adobe Photoshop/Illustrator
UX Skills Used:

Summary: I participated in a research project controlled by Mozilla and the University of Michigan. The overall aim of the project was to develop visions and future directions for the concept of a Mixed Reality internet browser - a browser that can intermix traditional 2-D internet content with Augmented and Virtual Reality content.

I worked on one question within this overall vision. How can we extend the specifications of CSS to control the look, behavior, and position of this AR content in an intelligent way?

My Contributions: I made 3 main contributions to this project:

  1. I developed the concept of the "Room Model" - a multi-faceted model for intelligent AR object placement.
  2. I developed a "playground" with which to explore the idea of controlling AR object layouts via simulated physics.
  3. I generated a view of how a future CSS specification might be written to cover both of the above concepts
